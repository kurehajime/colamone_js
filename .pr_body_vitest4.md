## 変更ログ / Breaking Changes 確認
- Migration Guide: https://vitest.dev/guide/migration

確認した主な破壊的変更:
- Vitest 4 は `Vite >= 6` と `Node.js >= 20` が必要
- `workspace` は `projects` へ移行
- coverage 関連オプションの一部削除

このリポジトリは `vite@^6.4.1` を使用しており、GitHub Actions も Node 20 で動作しています。また、Vitest の workspace / coverage 設定は使っていないため、今回の利用範囲では影響は限定的と判断しました。
