# How to play Colamone * Colamone is kinda like chess. Only some loose similarities. * Each piece can move 1 tile in the direction of any of it's dots. * If a piece gets to the other side, you get how many ever points it's worth. * If you get 8 points, you win. * When a piece gets to the other side, you can no longer move it and the enemy cannot take it. * When a player can't make any more moves, whoever has the highest score wins.
